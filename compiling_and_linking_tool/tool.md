# GNU binutils

## 簡介

binutils套件包含了以下工具：

* `ld`: GNU的連結器(linker)
* `as`: GNU的組合器(assembler)
* `addr2line`：將地址轉為檔名與行號
* `ar`: 建立、修改與存取檔案包(archive)的工具

* `strings`：列出檔案中可列印的字串。
* `strip`：去除檔案中的符號。
* `windmc`：相容Windows message編譯器。
* `windres`：Windows資源檔的編譯器。
